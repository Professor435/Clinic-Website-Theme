# Clinic-Website-Theme
**Astra Medical** is a modern, responsive healthcare website theme designed for advanced clinics and medical centers. 

**Key Features:**
- Clean, professional design with teal/green medical color palette
- Fully responsive layout optimized for all devices
- Interactive appointment booking modal with form validation
- Smooth scroll animations and hover effects
- Comprehensive sections: Hero, Services, Doctors, Testimonials, CTA
- Mobile-friendly navigation with hamburger menu
- Built with Tailwind CSS and vanilla JavaScript

**Highlights:**
- 6 medical service specialties with animated cards
- Doctor profiles with social links
- Patient testimonials with star ratings
- Statistics counters and trust badges
- Glass morphism effects and gradient backgrounds
- Functional contact form with success notifications

Perfect for hospitals, private clinics, dental practices, and specialist medical centers seeking a trustworthy, contemporary online presence.
